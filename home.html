HTML Questions

1.What is HTML?

HTML stands for HyperText Markup Language, used to structure content on the web.

2.Difference between HTML and XHTML?

XHTML is stricter (requires proper nesting, closing tags, and lowercase tags) and follows XML rules.

3.What are HTML tags and attributes?

Tags define elements (p, div).

Attributes provide extra info (href, id, class).Tags are enclosed in angle brackets, such as < p > for a paragraph or < h1 > for a heading.

4.What is the declaration?

Specifies the HTML version (e.g., DOCTYPE html for HTML5).

5.Explain semantic HTML.

Uses meaningful tags (header, article, footer) for better accessibility and SEO.

6.Difference between div and span?

div is a block-level element.span is inline.

7.What are void elements in HTML?

Elements with no closing tag (e.g., img, br, input).
